# Tasty Library
An alternative to Delicious Library.

```
usage: tl [-h] [-i] {connect,c,user,u,borrow,b,return,r,setup,book} ...

options:
  -h, --help            show this help message and exit
  -i, --interactive     Start in interactive mode

command:
  {connect,c,user,u,borrow,b,return,r,setup,book}
    connect (c)         Connect to a profile
    user (u)            Manage Users
    borrow (b)          Borrow books by user
    return (r)          return books
    setup               Setup new databases
    book                Manage Books
```
